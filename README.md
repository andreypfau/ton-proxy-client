# TON Proxy Client

## Build & Run

### macOS

1. Install Java using brew:
    ```shell
    brew install java
    ```

2. Run build script:

    ```shell
    ./build_macos.sh
    ```

3. Copy file `TON Proxy Client.app` to `/Applications`

4. Run & Enjoy visiting: [http://foundation.ton](http://foundation.ton)
